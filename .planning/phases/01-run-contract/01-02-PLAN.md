- Threat: `run_id` 或状态字段漂移，导致报告和执行记录无法关联。 Mitigation: 统一 `manifest.json` 和 `status.json` 必填字段，并写入 invariants。
- Threat: 恢复执行时错误切到新目录或新身份。 Mitigation: 要求 resumed run 复用相同 `run_id` 和 `artifact_root`。
</threat_model>
